AG Deliverables

The Accessibility Guidelines Working Group is currently working on Web Content Accessibility Guidelines (WCAG) 2.1. It also continues to maintain support materials for WCAG 2.0, including Understanding WCAG 2.0, WCAG 2.0 Techniques, and How to Meet WCAG 2.0. Editors' sources for the WCAG specifications are maintained in the W3C WCAG GitHub repository and the W3C WCAG 2.1 GitHub repository. GitHub users can watch this repository and send pull requests.

Deliverable Description Status Editors' Draft Formal Version
Web Content Accessibility Guidelines 2.1 Updates WCAG 2.0 with additional guidance to better address needs for users with low vision and cognitive disabilities, and users of touch and mobile interfaces. Success criteria proposals under review for First Public Working Draft. Editors' Draft of WCAG 2.1
Requirements for WCAG 2.1 Build on the existing requirements for WCAG 2.0, for WCAG 2.1. Being prepared for First Public Working Draft. Editors' Draft of WCAG 2.1 Requirements
Requirements for WCAG 2.0 Extensions Extensions are optional standards modules that build on the existing requirements for WCAG 2.0. Retired; AG WG is publishing WCAG 2.1 instead. Editors' Draft of Requirements for WCAG 2.0 Extensions Formal published version of Requirements for WCAG 2.0 Extensions
Understanding WCAG 2.0 Guide to understanding and using WCAG 2.0. Maintenance publications twice yearly. Editors' Draft of Understanding WCAG 2.0 Formal published version of Understanding WCAG 2.0
Techniques for WCAG 2.0 Provides techniques for web content authors and evaluators on meeting WCAG 2.0 Success Criteria. Maintenance publications twice yearly. Editors' Draft of WCAG 2.0 Techniques Formal published version of WCAG 2.0 techniques
Accessibility Conformance Testing Framework 1.0 Defines the format, logic, and implementation of accessibility test rules. Being developed by the ACT TF. Editors' Draft of Accessibility Conformance Testing Framework 1.0
Accessibility Conformance Testing Framework Requirements Outlines the requirements for the ACT Framework. Being developed by the ACT TF. Editors' Draft of Accessibility Conformance Testing Requirements
Cognitive Accessibility User Research Describes the challenges of using web technologies for people with learning disabilities or cognitive disabilities. Working Drafts under development by the COGA TF. Editors' Draft of Cognitive Accessibility User Research Formal published version of Cognitive Accessibility User Research
Issue Papers for the The Cognitive and Learning Disabilities Accessibility Task Force (COGA) Examines specific types of challenges in accessing web content for users with cognitive or learning disabilities. Working Drafts under development by the COGA TF. Editors' Draft of Issue Papers for the The Cognitive and Learning Disabilities Accessibility Task Force (COGA)
Cognitive Accessibility Roadmap and Gap Analysis Proposes how to enable accessibility for people with cognitive or learning disabilities. Working Drafts under development by the COGA TF. Editors' Draft of Cognitive Accessibility Roadmap and Gap Analysis

Previous work

The following deliverables were developed by the Working Group in the past, and have been completed.

Guidelines

  • How to meet WCAG 2.0
